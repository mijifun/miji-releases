# GezheStudio Desktop Application

> GezheStudio is a desktop application for AI music generation powered by the ACE-Step model.

## Downloads

Visit the [Releases](https://github.com/mijifun/miji-releases/releases) page to download the latest version.

## Supported Platforms

- **Linux** (Ubuntu, Debian, Fedora, openSUSE, and other distributions)
- **Windows** 10/11

## Installation

### Linux

Choose your preferred format:

#### .deb (Debian/Ubuntu)

```bash
sudo dpkg -i GezheStudio_*.deb
sudo apt-get install -f  # Install dependencies if needed
```

#### .rpm (Fedora/openSUSE)

```bash
sudo rpm -i GezheStudio-*.rpm
# or for Fedora
sudo dnf install GezheStudio-*.rpm
# or for openSUSE
sudo zypper install GezheStudio-*.rpm
```

### Windows

Download and run the installer:

- **GezheStudio-*-x64-setup.exe** - NSIS installer (recommended)
- **GezheStudio-*.msi** - MSI installer

## System Requirements

- **RAM**: 4GB minimum, 8GB recommended
- **Disk Space**: 500MB for application
- **Network**: Internet connection required for AI music generation

## Features

- Text-to-music generation using ACE-Step AI model
- Local processing - no server required for basic tasks
- Distributed computing support for complex compositions
- Real-time audio preview and export
